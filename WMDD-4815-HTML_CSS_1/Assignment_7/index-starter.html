Position Property

The position property is used to change the position of an element to something other than its default.

Beware: The position property should be used to achieve special effects and should not be used to create a grid layout or fix a broken layout.

Below is a summary of position values we will discuss in this class.

Position Property Values
Value Use Notes
relative Position an element relative to its normal position Use the CSS properties top, bottom, left, and right to adjust the position.
absolute Position an element relative to its nearest positioned parent. Use the CSS properties top, bottom, left, and right to adjust the position with respect to the parent. By "nearest positioned parent" what we mean is the ancestor element nearest in the hierarchy that has some position property set. (Usually just position:relative).
fixed Position an element relative to the viewport. Use the CSS properties top, bottom, left, and right to adjust the position with respect to the viewport.
sticky Position an element relative to the viewport when scrolling. Used to create a "sticky" element that stays in its normal place until you scroll by.

Tables

Tables are used to display tabular data; that is to say, data that is most easily understood in a table format. They should not be used to create grid layouts, since that violates the separation of style and content.

Ironically, you can find an example of a table in the above "Position" section.
